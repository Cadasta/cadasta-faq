Setting Basemap Transpancey

  1. FAQ
  2. Editing how the map looks
  3. Setting Basemap Transpancey

Purpose

This page will explain how to change the transparency of the basemap in a web map in ArcGIS Online

Requirements

  • You will need to have a username and password set up for you by a Cadasta team member. If you do not have a user please contact support@cadasta.org
  • You will need to sign to your account - see Introduction to your account for a reminder

Steps

  1. Navigate to the desired Web Map and press the "Item Details" button

  2. Click Open in Map Viewer

  3. See the Web Map with a visible background image (Basemap Image Layer)

  4. Click Content

  5. Click the Expand Arrow

    of the Basemap Imagery Layer Group to show the Basemap Layer(s)

    Note: This example uses the Dark Gray Canvas Basemap Imagery Layer Group, which includes the Dark Gray Reference and Dark Gray Base Layers.

  6. Click More Options

    found under the desired layer

    Note: This example shows the Dark Gray Base Layer.

  7. Click or hover your mouse over Transparency

  8. Set Transparency to 100% for any basemap imagery

  9. See that there is no longer a basemap present

  10. Click Save

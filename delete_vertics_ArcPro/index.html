Deleting Vertices in ArcGIS Pro

  1. FAQ
  2. Editing data
  3. Deleting Vertices in ArcGIS Pro
  1. In the "Map" tab, press the "Select" button and then select the polygon you wish to edit
  2. Select the "Edit" tab
  3. In the "Edit" tab, press the "Modify" button
  4. In the "Edit" tab -> "Tools" section, press the "Vertices" button
  5. Press the "Delete" button and the select the vertices in the map that you want to delete
  6. Press the "Finish" button

Deleting a Collected Survey Polygon

  1. FAQ
  2. Editing data
  3. Deleting a Collected Survey Polygon
  1. Navigate to your User Group

  1. Find the Field Collector Map (Web Map) you used to collect the polygons

  2. Select the Web Map

  3. Press the Open in Map Viewer button

  4. See the Map Viewer Overview

  5. Select Content

  6. See the Map Viewer Content Sidebar

  7. Hover the mouse over the polygon layer

  8. You will see multiple options. Select Show Table

  9. See Show Table view

  10. Click More Options within the table sub-window

  11. Select Show/Hide Columns

  12. You can toggle columns on and off by checking and unchecking the boxes next to the column names

    Note: In this example, only the OBJECTID column is toggled on and showing

    See when additional columns are toggled on

  13. Click More Options

  14. Select Center on Selection

    Note: The Selected Record in this example is OBJECTID 4

  15. See OBJECTID 4 Selection in Map Viewer

  16. Click on the selected polygon record to see the associated pop-up

  17. Select Edit in the pop-up to edit the polygon

  18. See the Edit pop-up window

  19. Press Delete to delete the selected polygon feature

  20. Press Close to stop editing

Creating a new feature layer

  1. FAQ
  2. Adding data
  3. Creating a new feature layer

Purpose

This page will explain how to create a new feature layer to be used in the Collector mobile app.

Requirements

  • You will need to have a username and passowrd set up for you by a Cadasta team member. If you do not have a user please contact support@cadasta.org

Steps

  1. Log in to your user account in AGOL.

  2. Navigate to Content, My Content

  3. Create a new Feature Layer by selecting Create

  4. Select Feature Layer

  5. Select Build a Layer

  6. Select Polygon and Create

  7. Select Next and Next
  8. Enter a Title and Tags. Other information is optional.
  9. Select Done
